# CascadeIQ
AI-powered Portfolio Risk Intelligence Platform using Neo4j, LangGraph, MCP, DORA Metrics, and Monte Carlo Simulation.



Features:
- Neo4j Dependency Graph
- Delay Propagation Analysis
- Risk Scoring Engine
- Monte Carlo Forecasting
- Resource Bottleneck Detection
- DORA Metrics
- Executive Narrative Generation
- LangGraph Workflow
- MCP Server Integration
- Streamlit Dashboards
